Intro to HiveMind

Thanks for playing HiveMind! This page has all the information needed to get started.

Making Words

The basic setup of HiveMind is that players are given a set of 7 letters. Players must then spell words (at least 4 letters long) using only those 7 letters, repeating letters if necessary. In addition, in each puzzle, one letter is given in yellow; this letter MUST be used at least once in every word spelled for that puzzle. We can see these rules in effect using the sample set of letters EGILNPS:
  • SPELLING, LEGS, ELLIPSE, and SLEEPLESSNESS would all be valid words.
  • PEN would be invalid (too short).
  • SLIPPING would be invalid (doesn't use the center letter).
  • GEEEEGGL would be invalid (not a word).
(Words are checked for validity using the Free Dictionary API).

Scoring

Words are scored depending on length:
  • 4-letter words are worth 1 point.
  • Words longer than 4 letters are scored at 1 point per letter.
A 7-point bonus is also given if the word is a "pangram" (i.e. uses all seven of the given letters). For example, going back to the example set of letters EGILNPS:
  • LEGS: 4 points.
  • ELLIPSE: 7 points.
  • SPELLING: 15 points (8 for length + pangram bonus of 7)
